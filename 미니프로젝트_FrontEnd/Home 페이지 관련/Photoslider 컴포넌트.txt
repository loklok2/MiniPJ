*****PhotoSlider 컴포넌트*****

1. PhotoSlider 컴포넌트:
 역할: 이 컴포넌트는 전달된 사진 목록에서 랜덤으로 선택된 세 개의 사진을 슬라이더 형식으로 표시하며, 
 사용자가  이미지를 클릭하거나 슬라이더 버튼을 통해 사진을 탐색할 수 있도록 합니다.

2. 상태 관리:
·currentIndex:
 역할: 현재 표시되고 있는 슬라이드의 인덱스를 관리합니다.
 설명: 사용자가 다음 또는 이전 버튼을 클릭할 때, 혹은 자동 슬라이딩이 작동할 때 인덱스가 변경됩니다.

·randomPhotos:
 역할: 전체 사진 목록에서 랜덤으로 선택된 세 개의 사진을 저장합니다. 
 설명: 처음에 전달된 사진 배열에서 무작위로 선택된 사진들이 이 상태에 저장되어 슬라이더에 표시됩니다.

·isClickable:
 역할: 사용자가 슬라이더 버튼을 클릭할 수 있는지 여부를 관리합니다. 
 설명: 슬라이드 전환 중에는 버튼 클릭을 비활성화하여 중복 클릭을 방지합니다.

·isImagesLoaded:
 역할: 슬라이더에 표시될 이미지들이 모두 로드되었는지 여부를 관리합니다.
 설명: 모든 이미지가 로드된 후에만 자동 슬라이딩 기능이 작동되도록 설정합니다.

·slideIntervalRef:
 역할: 자동 슬라이드 전환을 위한 인터벌 타이머를 저장합니다.
 설명: 자동 슬라이딩을 중지하거나 초기화할 때 사용합니다.

·location:
 역할: 현재 페이지의 위치 정보를 가져옵니다.
 설명: 사용자가 페이지를 떠나거나 뒤로 가기 버튼을 클릭할 때 슬라이더를 초기화하는 데 사용됩니다.

3. 함수 설명:
·selectRandomPhotos:
 역할: 전달된 사진 배열에서 무작위로 세 개의 사진을 선택합니다.
 설명: 이 함수는 슬라이더에 표시할 세 개의 사진을 무작위로 선택하여 슬라이더가 항상 새로운 사진을 보여줄 수 있도 	        록 합니다.

·handlePrevClick:
 역할: 사용자가 이전 버튼을 클릭했을 때 슬라이더의 인덱스를 감소시킵니다. 
 설명: 클릭 가능한 상태에서만 작동하며, 현재 인덱스가 0일 경우 마지막 인덱스로 순환합니다.

·handleNextClick:
 역할: 사용자가 다음 버튼을 클릭했을 때 슬라이더의 인덱스를 증가시킵니다. 
 설명: 클릭 가능한 상태에서만 작동하며, 현재 인덱스가 마지막 인덱스일 경우 첫 번째 인덱스로 순환합니다.

·handlePhotoClick:
 역할: 사용자가 슬라이더 이미지를 클릭했을 때 해당 이미지의 URL을 부모 컴포넌트에 전달합니다. 
 설명: 클릭 가능한 상태에서만 작동하며, 클릭 후 자동 슬라이딩을 중지합니다.

·handleImageLoad:
 역할: 이미지가 모두 로드되었을 때 로드 상태를 true로 설정합니다.
 설명: 모든 이미지가 로드되면 자동 슬라이딩 기능이 활성화됩니다.

4. UI 구성:
·이전/다음 버튼:
 설명: SlArrowLeft와 SlArrowRight 아이콘을 사용하여 슬라이더의 이전 및 다음 이미지를 탐색할 수 있는 버튼을 제공합        니다. 
        버튼은 화면 중앙 좌우에 위치하며, 배경색이 어두운 반투명으로 설정되어 있습니다.

·이미지 슬라이드:
 설명: 랜덤으로 선택된 이미지가 슬라이더에 표시되며, 현재 인덱스에 해당하는 이미지만 보이도록 설정됩니다. 
        이미지가 전환될 때 부드럽게 페이드 인/아웃 효과를 적용합니다.

*****Tailwind CSS 클래스 상세 설명*****

1.relative
  설명: 이 클래스는 요소를 상대적으로 배치하여 
         자식 요소들이 해당 요소를 기준으로 절대 위치를 가질 수 있게 합니다.
  예시: <div className="relative w-full h-[400px]">
  결과: 해당 div는 자식 요소들이 absolute 클래스를 사용할 때 이 요소를 기준으로 배치됩니다.

2. w-full
  설명: 이 클래스는 요소의 너비를 부모 컨테이너의 100%로 설정합니다.
  예시: <div className="w-full">
  결과: 해당 div 요소는 부모의 전체 너비를 차지합니다.

3. h-[400px]
  설명: 이 클래스는 요소의 높이를 400px로 고정 설정합니다.
  예시: <div className="h-[400px]">
  결과: 해당 div 요소의 높이가 400px로 설정됩니다.

4. overflow-hidden
  설명: 이 클래스는 요소의 콘텐츠가 요소의 영역을 벗어날 때 보이지 않도록 설정합니다.
  예시: <div className="overflow-hidden">
  결과: 해당 div 요소의 영역을 벗어나는 콘텐츠는 숨겨집니다.

5. absolute
  설명: 이 클래스는 요소를 절대 위치로 배치하여 부모 요소를 기준으로 위치를 지정할 수 있습니다.
  예시: <button className="absolute top-1/2 left-4">
  결과: 해당 버튼은 부모 요소를 기준으로 좌측에서 4px, 상단에서 50% 위치에 배치됩니다.

6. top-1/2
  설명: 이 클래스는 요소의 상단 위치를 부모 컨테이너의 50%로 설정합니다.
  예시: <button className="top-1/2">
  결과: 해당 버튼은 부모 요소의 상단에서 50% 위치에 배치됩니다.

7. transform -translate-y-1/2
  설명: 이 클래스는 요소를 상하 방향으로 -50%만큼 이동시켜 요소의 세로축 중앙에 위치하도록 합니다.
  예시: <button className="transform -translate-y-1/2">
  결과: 해당 버튼은 상단에서 50%만큼 이동한 후, 
         다시 -50%만큼 올라와 세로축의 정중앙에 배치됩니다.

8. bg-black bg-opacity-50
  설명: 이 클래스는 요소의 배경색을 검은색으로 설정하며, 불투명도를 50%로 설정하여 반투명 효과를 줍니다.
  예시: <button className="bg-black bg-opacity-50">
  결과: 해당 버튼의 배경이 반투명한 검은색으로 설정됩니다.

9. text-white
  설명: 이 클래스는 요소의 텍스트 색상을 흰색으로 설정합니다.
  예시: <button className="text-white">
  결과: 해당 버튼의 텍스트가 흰색으로 표시됩니다.

10. p-2
  설명: 이 클래스는 요소의 모든 면에 0.5rem(8px) 크기의 패딩을 추가합니다.
  예시: <button className="p-2">
  결과: 해당 버튼의 모든 면에 0.5rem 크기의 패딩이 적용됩니다.

11. rounded-full
  설명: 이 클래스는 요소를 완전히 둥글게 만들어 원형으로 보이게 합니다.
  예시: <button className="rounded-full">
  결과: 해당 버튼이 원형 모양으로 표시됩니다.

12. z-10
  설명: 이 클래스는 요소의 z-index를 10으로 설정하여 다른 요소들보다 앞쪽에 표시되도록 합니다.
  예시: <button className="z-10">
  결과: 해당 버튼이 다른 요소들보다 위에 표시됩니다.

13. object-cover
  설명: 이 클래스는 이미지가 요소의 크기에 맞게 잘리더라도 비율을 유지하며 꽉 채워지도록 합니다.
  예시: <img className="object-cover">
  결과: 해당 이미지가 요소의 크기에 맞게 채워지면서도 비율을 유지합니다.

14. transition-opacity duration-1000
  설명: 이 클래스는 요소의 불투명도가 변경될 때 1초 동안 부드럽게 전환되도록 설정합니다.
  예시: <img className="transition-opacity duration-1000">
  결과: 해당 이미지의 불투명도가 1초에 걸쳐 부드럽게 변경됩니다.

15. opacity-100
  설명: 이 클래스는 요소의 불투명도를 100%로 설정하여 완전히 보이도록 합니다.
  예시: <img className="opacity-100">
  결과: 해당 이미지가 완전히 보입니다.

16. opacity-0
  설명: 이 클래스는 요소의 불투명도를 0%로 설정하여 완전히 보이지 않도록 합니다.
  예시: <img className="opacity-0">
  결과: 해당 이미지가 완전히 투명해져 보이지 않습니다.
